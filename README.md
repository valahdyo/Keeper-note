# Keeper-note

## General info
This project is a google keep clone.
	
## Technologies
Project is created with:
* Node.js
* React.js
* Material UI
	
## Setup
To run this project, install it locally using npm:
```
$ cd /@where-you-save-repo/
$ npm install
$ npm start
```

Or, you can see my project at: https://keeper-note21.herokuapp.com/
